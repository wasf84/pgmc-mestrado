Elastic Net é um método de regressão que executa, simultaneamente à seleção de variáveis, um processo de regularização. Regularização é um procedimento para quando o modelo está overfitting, ou seja, o modelo está muito bem ajustado para os dados de treino mas não gera bons resultados com os dados de teste.

A regularização pode ser realizada de duas maneiras: utilizando penalização l1 ou penalização l2. A penalização l1 é chamada de regressão 'lasso' e a penalização l2 é a chamada regressão 'ridge'.

A regressão lasso tenta ajustar os coeficientes do modelo para zero deixando apenas um pequeno subconjunto diferentes de zero. Por sua vez, a penalização ridge não busca ajustar os coeficientes para 0, contudo, o método encolhe os coeficientes o mais próximo possível de 0. Desta forma a regressão ridge não remove os preditores na seleção do modelo.

============

Elastic Net is a regression method that performs, simultaneously with the selection of variables, a regularization process. Regularization is a procedure for when the model is overfitting, that is, the model is very well adjusted to the training data but does not generate good results with the test data.

The regularization can be performed in two ways: using penalty l1 or penalty l2. Penalty l1 is called 'lasso' regression and penalty l2 is called 'ridge' regression.

Lasso regression tries to adjust the model coefficients to zero leaving only a small subset non-zero. In turn, the ridge penalty does not seek to adjust the coefficients to 0, however, the method shrinks the coefficients as close as possible to 0. Thus, the ridge regression does not remove the predictors in the model selection.

============

A regressão Lasso é um método de regressão linear que adiciona uma penalização à soma dos valores absolutos dos coeficientes do modelo, a fim de reduzir o efeito de variáveis irrelevantes e evitar o overfitting. Esta penalização é controlada por um parâmetro de regularização, que é ajustado para encontrar o equilíbrio adequado entre ajuste ao conjunto de dados e simplicidade do modelo. Em resumo, a regressão Lasso é uma técnica de seleção de variáveis que ajuda a identificar quais variáveis são mais relevantes para prever uma determinada resposta.

============

A regressão Ridge é um método de regressão linear que adiciona uma penalização à soma dos quadrados dos coeficientes do modelo, a fim de reduzir a variância dos coeficientes e evitar o overfitting. Esta penalização é controlada por um parâmetro de regularização, que é ajustado para encontrar o equilíbrio adequado entre ajuste ao conjunto de dados e simplicidade do modelo. Em resumo, a regressão Ridge é uma técnica de regularização que ajuda a reduzir o efeito de multicolinearidade e a melhorar a estabilidade dos coeficientes de um modelo de regressão linear.
